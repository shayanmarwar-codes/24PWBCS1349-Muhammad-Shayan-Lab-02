# Lab 2 - Web Technologies

This repository contains Lab 2 exercises for the CS 224L Web Technologies course:

- Task 1: Personal Profile Page (profile.html)
- Task 2: Course Schedule (schedule.html)
- Task 3: Navigation Menu (added to all pages)
- Task 4: Registration Form (registration.html)
- Task 5: GitHub Pages Deployment
- Task 6: Landing Page (index.html)
- Task 7: Cross-Page Navigation

All pages are linked via a consistent navigation menu.  
This repository demonstrates HTML structure, tables, forms, navigation, and GitHub Pages deployment.
